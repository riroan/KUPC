# ucpc2020-description-layout

Based on:

* [olymp.sty](https://github.com/GassaFM/olymp.sty/blob/master/LICENSE)
* [SPC 2019](https://acm.sogang.ac.kr/spc/) 문제지 레이아웃

문제 본문과 예제의 저작권은 각 문제 작성자에게 있습니다.

## 테마 사용 예

* [UCPC 2020](main.pdf)
* [SUAPC 2020](https://upload.acmicpc.net/5286c73a-4429-4869-bf3c-be62ac9d4714/)

## 참고

- **X<sub>E</sub>L<sub>A</sub>T<sub>E</sub>X을 사용해 컴파일해야 합니다.**
- 원본 솔루션과 약간의 차이가 있습니다.
  - 원본 솔루션의 삼세리프 폰트는 [Avenir Next](https://www.fonts.com/font/linotype/avenir-next)라는 유료 폰트로 개인적으로 라이센스를 소유하고 있기 때문에, [Noto Sans CJK](https://www.google.co.kr/get/noto/)로 교체했습니다.
- 커스텀 커맨드가 있습니다.
  - `\t{text}`: `text`를 고정폭의 붉은 글씨로 작성합니다. UCPC 컨벤션에서 본문과 명확히 구분되는 리터럴일 때 사용합니다.

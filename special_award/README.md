## 특별상

## 파일 구조

### json

- award_slide.json : 시상때 보일 장면

- contest_info.json, contest.json : 사용자에 대한 정보

- final_runs.json : 최종 스코어보드 파일

### python

- get_rank.py : 순위 매기기

- award.py : 특별상 뽑는 과정

- run.py : 순위 매기고 특별상 뽑고 award_slide.json에 저장

- special_award.py : 특별상 뽑는 로직

### text

- special_award.txt : 특별상 목록
